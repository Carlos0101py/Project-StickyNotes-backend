Criar Migrações:
Depois de configurar a extensão Flask-Migrate, você pode criar migrações para
o banco de dados sempre que houver alterações nos modelos. Isso é feito executando
o seguinte comando no terminal: flask db init
Isso criará um diretório de migrações no seu projeto.

Aplicar Migrações:
Depois de criar as migrações, você pode aplicá-las ao banco de 
dados com o seguinte comando: flask db migrate
Isso irá gerar arquivos de migração com base nas alterações nos modelos.

Atualizar o Banco de Dados:
Por fim, você pode aplicar as migrações ao banco de dados com 
o seguinte comando: flask db upgrade
Isso aplicará as migrações pendentes ao banco de dados.
